# Stylus skeleton

Must be used with PostCSS plugins: `autoprefixer`, `rucksack-css`, `cssnano`.

## Structure
- Main (with descriptions)
    - Settings
    - Functions (and mixins)
        - Spacing
        - Resets
    - Base (without prefixes)
        - resets
        - document
        - typografy
    - Layout (`.l-`,`.o-` prefixes) 
        - Grid
        - Media
    - Components (`.c-` prefix)
    - Utils (`.u-` prefix)
        - Responsive
        - Overflow
        - Text
        - Hacks
        - Typography
        - Padding
        - Margin
        - Display
        - Clearfix
